cleaner:pencil
Master Pencil

Required Privileges:

    server

Image:

Usage:

    place (right-click):
- when not pointing at a node, changes modes
- when pointing at a node, sets node to be used

use (left-click):
- executes action for current mode:
  - erase: erases pointed node
  - write: adds node
  - swap:  replaces pointed node
